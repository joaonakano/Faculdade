# Como rodar o código com o XAMPP?
- Basta **executar o MySQL e Apache Server** pelo Painel do XAMPP e criar uma nova database chamada `aula-12`.

![PhpMyAdmin](https://github.com/joaonakano/Faculdade/blob/main/Desenvolvimento-De-Sistemas-Web-Mobile-III/bimestres/2-bimestre/aula-12-mao-na-massa/assets/phpmyadmin-db.png?raw=true "PhpMyAdmin")

- Clonar este repositório, acessá-lo por algum Console ou Terminal, digitar ``composer install`` e aguardar a instalação das dependências.

![Instalação](https://github.com/joaonakano/Faculdade/blob/main/Desenvolvimento-De-Sistemas-Web-Mobile-III/bimestres/2-bimestre/aula-12-mao-na-massa/assets/installation.png?raw=true "Instalação")

- Executar o comando ``php artisan serve`` para iniciar a aplicação Laravel.

![running](https://github.com/joaonakano/Faculdade/blob/main/Desenvolvimento-De-Sistemas-Web-Mobile-III/bimestres/2-bimestre/aula-12-mao-na-massa/assets/run.png?raw=true "running")

- Acessar as rotas `/salas` e `/reservas`!
